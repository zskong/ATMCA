> Official code implementation for: **"ATMCA: Augmented Tensorized Consensus Learning for Multi-view Clustering with Anchor-Aligned"**. This paper has been accepted by *IEEE Transactions on Multimedia (TMM) 2026*. 

## 🌟 Overview

We propose a novel method that aims to resolve the representation confusion caused by anchor mismatch!

### 🔑 Key Highlights
* **New insight**: We provide an intuitive solution to the Anchor-Unaligned Problem.
* **Augmented**: We use V+1 views instead of V views to capture the consistent information.
